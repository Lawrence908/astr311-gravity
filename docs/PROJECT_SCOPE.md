# Project Scope

- **Path A – Cosmic structure (primary):**
  - Toy N-body gravity in 2D → 3D.
  - Demonstrates how small density perturbations grow into clustered structure (proto-galaxies).

- **Path B – Colliders (extension):**
  - Simple numerical models for collider size vs energy.
  - Connects collider concepts back to early-universe conditions.

Constraints:
- Educational, astronomy-focused, and honest about assumptions.
- Prefer clear visuals and stable demos over realism.
- CPU reference first; GPU or advanced techniques only as optional stretch work.
