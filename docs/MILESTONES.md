# Milestones (high level)

- **Bootstrap**
  - Repo initialized.
  - `python -m main` (from `src/`) prints a friendly message.

- **Path A (2D CPU reference)**
  - Implement 2D particle state and initial conditions.
  - Implement clear O(N²) gravity on CPU with softening.
  - Implement a basic integrator and run a tiny test.

- **Visualization**
  - Minimal live 2D scatter plot.
  - Ability to export frames to `outputs/frames/`.

- **Validation**
  - Basic stability checks on trajectories.
  - Visual confirmation that clustering behavior looks plausible.

- **Path B (Week 1–2)**
  - Define a simple collider model and radius–energy scaling.
  - One basic visualization or text summary.

- **Stretch**
  - Optional GPU or performance work.
  - Optional 3D extension for Path A.
