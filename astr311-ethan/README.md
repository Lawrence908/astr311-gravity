# How to run

From same directory as this README:
- `source .venv/bin/activate`
- `cd backend`
- `uvicorn main:app --reload`

# Open browser to:
# - Dashboard: http://localhost:8000/
# - Three-body: http://localhost:8000/frontend/index.html?scenario=three_body
# - Pluto: http://localhost:8000/frontend/index.html?scenario=pluto_system

Only pluto and 3 body problem are working, other options on the dashboard are just filler
